# cake-demo

Учебный проект по предметной области **торты**.

## Что есть в проекте

- вход в систему по логину и паролю
- вход как гость
- просмотр каталога тортов
- поиск товаров
- сортировка товаров
- просмотр заказов
- добавление, редактирование и удаление товаров для администратора
- работа с базой данных PostgreSQL

## Что использовалось

- Python 3.11
- Flask
- PostgreSQL
- pgAdmin 4
- HTML
- CSS

## Структура проекта

- `app` - основная логика приложения
- `templates` - html шаблоны
- `static` - стили и изображения
- `db` - sql файл для структуры базы данных
- `import` - файлы для импорта данных

## Как запустить проект

1. Создать базу данных `cake_demo` в PostgreSQL.
2. Установить зависимости:

```bash
py -3.11 -m pip install -r requirements.txt
Выполнить импорт данных:
py -3.11 import_data.py
Запустить проект:
py -3.11 run.py
Открыть в браузере:
http://127.0.0.1:5000


Примечание

Для работы проекта нужно, чтобы PostgreSQL был запущен, а в файле подключения к базе были указаны правильные данные пользователя и пароль.
